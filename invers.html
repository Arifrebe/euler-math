  • Jika f'(b)=0, turunan invers tidak terdefinisi

Penjelasan Edukatif

Apa Itu Fungsi Invers?

Fungsi invers adalah fungsi yang membalikkan proses dari fungsi asal. Jika sebuah fungsi mengubah x menjadi y, maka fungsi invers akan mengubah y kembali menjadi x.

Secara matematis:

Jika f(x) = y, maka f⁻¹(y) = x

Contoh sederhana:

  • f(x) = x + 3 → menambah 3
  • f⁻¹(x) = x − 3 → mengurangi 3 (membalikkan)

Fungsi invers hanya ada jika fungsi tersebut bijektif, artinya setiap input memiliki satu output unik dan sebaliknya.

Cara Mencari Fungsi Invers

Langkah umum mencari invers f(x):

  1. Ganti f(x) dengan y
  2. Tukar posisi x dan y
  3. Selesaikan persamaan untuk y
  4. Ganti y dengan f⁻¹(x)

Contoh:

f(x) = 2x + 5

  1. y = 2x + 5
  2. Tukar → x = 2y + 5
  3. Selesaikan untuk y → y = (x - 5) / 2
  4. Maka f⁻¹(x) = (x - 5) / 2

Turunan Fungsi Invers

Turunan invers tidak selalu dihitung langsung. Ada rumus khusus yang memudahkan:

\((f^{-1})'(x) = \dfrac{1}{f'(f^{-1}(x))}\)

Artinya:

  • Kita ambil turunan fungsi asli: f'(x)
  • Kemudian masukkan nilai f⁻¹(x) ke dalam turunan tersebut
  • Lalu ambil kebalikan (1 dibagi hasilnya)

Contoh Turunan Fungsi Invers

Diberikan: f(x) = 3x + 4

1. Cari invers: f⁻¹(x) = (x - 4) / 3

2. Turunan fungsi asli: f'(x) = 3

3. Turunan invers:

\((f^{-1})'(x) = \dfrac{1}{f'(f^{-1}(x)) = 3}\)

Hasilnya konstan karena f adalah fungsi linear.

Ringkasan

  • Fungsi invers membalikkan fungsi asli
  • Invers hanya ada jika fungsi bijektif
  • Turunan invers mengikuti rumus: (f⁻¹)'(x) = 1 / f'(f⁻¹(x))
  • Digunakan dalam kalkulus, transformasi, dan analisis fungsi
